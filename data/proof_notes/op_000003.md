# Problem op_000003

## Statement
∀ {α : Type u} [SemigroupLike01 α], ∃ e : α, ∀ x : α, e * x = x ∧ x * e = x

## Attempt Result
counterexample

## Natural Language Sketch
反例として `ULift Bool` に左射影演算 `x * y := x` を入れる。このとき 3 公理はすべて `rfl` で成り立つ。もし両側単位元 `e` があれば、左単位条件 `e * x = x` を `x = true, false` に適用して、それぞれ `e = true` と `e = false` を得るので矛盾する。したがって一般にはそのような `e` は存在しない。

## Counterexample Intuition
反例は `ULift Bool` 上の演算 `x * y = x` である。これは常に左の引数を返すので、与えられた 3 つの公理は満たす。一方、もし全ての `x` に対して `e * x = x` を満たす元 `e` があるとすると、`x = true` から `e = true`、`x = false` から `e = false` となり矛盾する。したがって両側単位元の存在は一般には成り立たない。
