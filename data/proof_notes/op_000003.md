# Problem op_000003

## Statement
∀ {α : Type u} [SemigroupLike01 α], ∃ e : α, ∀ x : α, e * x = x ∧ x * e = x

## Attempt Result
counterexample

## Natural Language Sketch
反例として `Fin 3` 上に `x * y := if x = y then x else 2` を入れる。これは `{0,1}` の上に頂点 `2` を足した 3 元 join 半束で、与えられた 3 公理は有限個のケース分けで満たされる。だが単位元は存在しない。実際、`e = 0` は `x = 1` で失敗し、`e = 1` は `x = 0` で失敗し、`e = 2` は `x = 0` で失敗する。したがって全ての `SemigroupLike01` に単位元があるという主張は偽。

## Counterexample Intuition
`α = Fin 3` とし、`x * y` を `x = y` のときは `x`、そうでなければ `2` と定める。すると `0 * 1 = 2`, `1 * 0 = 2`, `0 * 2 = 2`, `2 * 0 = 2` などとなり、`2` は頂点の役割をする。この演算は 3 公理を満たすが、単位元はない。候補 `e = 0` は `x = 1` で `e * x = 2 ≠ 1`、`e = 1` は `x = 0` で失敗し、`e = 2` は `x = 0` で `e * x = 2 ≠ 0` となる。
