# Problem op_000019

## Statement
∀ α : Type u, [Nontrivial α], ∃ _ : SemigroupLike01 α, ∃ x y : α, x * y = x ∧ y * x = y ∧ x ≠ y

## Attempt Result
proof

## Natural Language Sketch
既存定理 `thm_op_000016` を使って、任意の `α` に left-projection 型の `SemigroupLike01` 構造 `x * y = x` を入れる。`Nontrivial α` から `exists_pair_ne α` で異なる `x,y` を取れば、`x * y = x` と `y * x = y` はどちらもその構造の性質 `hmul` から直ちに従う。

## Counterexample Intuition
(empty)
