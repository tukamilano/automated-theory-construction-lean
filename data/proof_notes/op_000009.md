# Problem op_000009

## Statement
∀ {α : Type u} [SemigroupLike01 α], ∀ x y : α, (x * y) * x = x * y

## Attempt Result
proof

## Natural Language Sketch
`ax_middle_swap` に `z := x` を入れると `(x * y) * x = (x * x) * y`。さらに `ax_left_idempotent` から `x * x = x` なので、右辺は `x * y` に簡約される。

## Counterexample Intuition
(empty)

## Lean Draft
```lean
intro α _ x y
have hswap : (x * y) * x = (x * x) * y := by
  simpa [op] using SemigroupLike01.ax_middle_swap (α := α) x y x
have hxx : x * x = x := by
  simpa [op] using SemigroupLike01.ax_left_idempotent (α := α) x
simpa [hxx] using hswap
```
