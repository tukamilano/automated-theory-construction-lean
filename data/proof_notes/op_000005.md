# Problem op_000005

## Statement
∀ {α : Type u} [SemigroupLike01 α], ∀ join : α → α → α, ∀ x y z : α, x * (join y z) = join (x * y) (x * z)

## Attempt Result
counterexample

## Natural Language Sketch
既存の `Scratch.lean` と同型の 3 要素モデルを使う。`α := ULift (Fin 3)`、`top := 2`、`op3 x y := if x = y then x else top` とおくと、`SemigroupLike01` の 3 公理は計算で成り立つ。そこで `join0` を常に `0` を返す定数関数にし、`x = 1, y = 0, z = 1` を代入する。仮定は `one * join0 zero one = join0 (one * zero) (one * one)` を与えるが、`simp` すると `top = zero` になり矛盾する。

## Counterexample Intuition
`join` が完全に任意なので成り立たない。3 要素集合 `ULift (Fin 3)` 上で、積を `x = y` なら `x`、そうでなければ `top` を返す演算にすると `SemigroupLike01` 公理は満たされる。ここで `join` を常に `0` を返す定数関数に取り、`x = 1, y = 0, z = 1` とすると、左辺は `1 * 0 = top`、右辺は `join (1 * 0) (1 * 1) = 0` となって一致しない。
