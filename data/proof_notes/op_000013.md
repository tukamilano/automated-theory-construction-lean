# Problem op_000013

## Statement
∃ α : Type u, ∃ _ : SemigroupLike01 α, ∃ x y : α, x * y = x ∧ x ≠ y

## Attempt Result
proof

## Natural Language Sketch
`α := ULift Bool` に左射影演算 `x * y := x` を入れる。すると `SemigroupLike01` の3公理はすべて定義通りに `rfl` で成り立つ。あとは `x := ULift.up true`, `y := ULift.up false` を取れば `x * y = x` は自明で、`x ≠ y` は `ULift.down` を作用させて `true ≠ false` に帰着する。

## Counterexample Intuition
(empty)
