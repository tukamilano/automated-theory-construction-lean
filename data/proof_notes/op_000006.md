# Problem op_000006

## Statement
∀ {α : Type u} [SemigroupLike01 α], ∀ x y z : α, x * y = x * z → y = z

## Attempt Result
counterexample

## Natural Language Sketch
反例は `α := ULift Bool` と左射影演算 `x * y := x`。このとき `ax_left_idempotent`, `ax_right_absorb_duplicate`, `ax_middle_swap` はすべて `rfl` で成り立つ。すると常に `x * y = x * z` なので、仮に左消去 `x * y = x * z → y = z` が全 `SemigroupLike01` で成り立つなら、`x = ⟨true⟩, y = ⟨true⟩, z = ⟨false⟩` を代入して `⟨true⟩ = ⟨false⟩` を得て矛盾。

## Counterexample Intuition
`ULift Bool` 上で演算を左射影 `x * y := x` とする。この構造は 3 つの公理を満たすが、`x * y` は右引数を完全に無視するので `x * y = x * z` が常に成り立つ。したがって `x = true, y = true, z = false` では前件は成り立つ一方、結論 `y = z` は偽であり、左消去則は成り立たない。
