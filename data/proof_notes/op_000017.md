# Problem op_000017

## Statement
∀ α : Type u, [Nontrivial α], ∃ _ : SemigroupLike01 α, ∃ x y : α, x * y = x ∧ x ≠ y

## Attempt Result
proof

## Natural Language Sketch
任意の非自明型 α に対し、積を左射影 `x * y := x` で定める。このとき `x * x = x`, `x * (x * y) = x * y`, `(x * y) * z = (x * z) * y` はすべて定義展開で `rfl`。よって `SemigroupLike01 α` が入る。さらに `Nontrivial α` から `exists_pair_ne α` で異なる 2 点 `x ≠ y` を取り、左射影より `x * y = x`。これで所望の `x, y` が得られる。

## Counterexample Intuition
(empty)
