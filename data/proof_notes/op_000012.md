# Problem op_000012

## Statement
∃ α : Type u, ∃ _ : SemigroupLike01 α, ∃ x y z : α, x * y = x * z ∧ y ≠ z

## Attempt Result
proof

## Natural Language Sketch
具体モデルとして `α = ULift Bool` に左射影演算 `x * y = x` を入れる。このとき 3 つの公理はすべて `rfl` で成り立つ。さらに `x = true`, `y = true`, `z = false` を取れば `(x * y) = x = (x * z)` で、`y ≠ z` は `ULift.down` を取って `true ≠ false` に帰着する。

## Counterexample Intuition
(empty)
