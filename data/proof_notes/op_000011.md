# Problem op_000011

## Statement
∃ α : Type u, ∃ _ : SemigroupLike01 α, ¬ Subsingleton α

## Attempt Result
proof

## Natural Language Sketch
台集合として `α := ULift Bool` を取り、演算を左射影 `x * y := x` と定める。すると `x * x = x`, `x * (x * y) = x * y`, `(x * y) * z = (x * z) * y` はすべて定義通りで `rfl`。もし `Subsingleton α` なら、持ち上げた `true` と `false` が等しくなるが、`ULift.down` を作用させると `true = false` となり矛盾。

## Counterexample Intuition
(empty)

## Lean Draft
```lean
by
  let α : Type u := ULift.{u} Bool
  letI : Mul α := ⟨fun x _ => x⟩
  refine ⟨α, ?_⟩
  refine ⟨{ mul := fun x _ => x
    ax_left_idempotent := by
      intro x
      rfl
    ax_right_absorb_duplicate := by
      intro x y
      rfl
    ax_middle_swap := by
      intro x y z
      rfl }, ?_⟩
  intro h
  have hEq : ((⟨true⟩ : α)) = (⟨false⟩ : α) := h.elim _ _
  exact Bool.true_ne_false (congrArg ULift.down hEq)
```
