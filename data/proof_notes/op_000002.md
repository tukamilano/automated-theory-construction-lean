# Problem op_000002

## Statement
∀ {α : Type u} [SemigroupLike01 α], ∀ x y : α, x * y = y * x

## Attempt Result
counterexample

## Natural Language Sketch
反例。α = Bool に左射影演算 x * y := x を入れると、x * x = x, x * (x * y) = x * y, (x * y) * z = (x * z) * y はすべて定義展開で rfl。だが true * false = true かつ false * true = false なので可換ではない。Lean ではこの局所インスタンスを `letI` で入れ、仮定を Bool 上の `true false` に適用して `true = false` を得て `cases` で矛盾にする。

## Counterexample Intuition
Bool 上で x * y = x と定める。すると各公理はどの式も左引数に簡約されるので成立する。しかし true * false = true で、false * true = false だから、一般には x * y = y * x は成り立たない。
